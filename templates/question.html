InfinitePay QA

About

This a proof of concept QA tool for InfinitePay.

It's powered by OpenAI's Embeddings API and built in Python with LangChain and Faiss.

How it works

  • All articles from InfinitePay's Help Center are scraped.
  • Embeddings are generated with OpenAI's API and passed to Faiss.
  • The embedding vector and Faiss index are store in local files.
  • When a user asks a question, the most similar documents to the question are found and used as context for OpenAI's prompt.
